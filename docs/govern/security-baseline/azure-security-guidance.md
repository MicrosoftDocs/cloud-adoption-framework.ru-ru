# <a name="microsoft-security-guidance"></a>Руководство по безопасности Майкрософт

## <a name="tools"></a>Инструменты

[Портал доверия служб Майкрософт](https://servicetrust.microsoft.com) и менеджер по соответствию требованиям могут помочь удовлетворить эти потребности.

- Преодолеть проблемы управления соответствием.
- Ответственность за соблюдение нормативных требований.
- Проводить самостоятельное аудит и оценивать риски использования облачной службы предприятия.

Эти средства предназначены для того, чтобы помочь организациям удовлетворить сложные обязательства по обеспечению соответствия и улучшить возможности защиты данных при выборе и использовании облачных служб Майкрософт.

**Портал доверия служб Майкрософт** предоставляет подробные сведения и средства, которые помогут удовлетворить ваши потребности в использовании облачных служб Майкрософт, включая Azure, Microsoft 365, Dynamics 365 и Windows. Портал — это один и тот же магазин для обеспечения безопасности, нормативных сведений, соответствия требованиям и конфиденциальности, относящийся к Microsoft Cloud. Здесь публикуются сведения и ресурсы, необходимые для выполнения оценки риска самообслуживания облачных служб и средств. Портал был создан для помощи в отслеживании нормативных требований в Azure, в том числе:

- **Менеджер по соответствию требованиям:** Диспетчер соответствия требованиям. средство оценки рисков на основе рабочих процессов на портале доверия служб Майкрософт позволяет отслеживанию, назначение и проверку нормативных требований Организации, связанных с облачными службами Майкрософт, такими как Microsoft 365, Dynamics 365 и Azure. Дополнительные сведения см. в следующем разделе.
- **Доверять документам:** Три категории руководств предоставляют устойчивый ресурсы для оценки Microsoft Cloud, сведения о Microsoft Operations в обеспечении безопасности, соответствии требованиям и конфиденциальности, а также помогут вам улучшить возможности защиты данных. К этим руководствам относятся:
- **Отчеты об аудите:** Отчеты аудита позволяют поддерживать актуальность сведений о конфиденциальности, безопасности и соответствии требованиям для облачных служб Майкрософт. Эти сведения включают в себя ISO, SOC, FedRAMP и другие отчеты об аудите, письма моста и материалы, связанные с независимыми сторонними аудитами облачных служб Майкрософт, таких как Azure, Microsoft 365, Dynamics 365 и др.
- **Руководства по защите данных:** Руководства по защите данных предоставляют сведения о том, как облачные службы Майкрософт защищают ваши данные и как можно управлять безопасностью и соответствием облачных данных в Организации. В этих руководствах содержатся подробные технические документы о проектировании и работе облачных служб Майкрософт, документов о часто задаваемых вопросах, об оценках безопасности в конце года, результатах тестирования на проникновение и руководствах, которые помогут вам провести оценку рисков и улучшить возможности защиты данных.
- Схема **безопасности и соответствия требованиям Azure:** Схемы предоставляют ресурсы, помогающие в создании и запуске облачных приложений, которые помогут обеспечить соответствие строгим нормативам и стандартам. Мы имеем больше сертификатов, чем любой другой поставщик облачных служб, поэтому вы можете уверенно развертывать критически важные рабочие нагрузки в Azure с помощью схем, которые включают в себя:
  - Обзор и руководство для конкретной отрасли.
  - Матрица обязанностей клиента.
  - Эталонные архитектуры с моделями угроз.
  - Управляющие матрицы реализации.
  - Автоматизация для развертывания эталонных архитектур.
  - Ресурсы о конфиденциальности. Документация по оценкам влияния на защиту данных, запросам на тему данных и уведомлению об обнаружении данных предоставляется для внедрения в собственную программу поддержки Общий регламент по защите данных (GDPR).
- **Начало работы с GDPR:** Продукты и службы Майкрософт помогают организациям удовлетворить требования GDPR при сборе или обработке персональных данных. Портал доверия служб Майкрософт предназначен для предоставления информации о возможностях служб Майкрософт, которые можно использовать для решения конкретных требований GDPR. Благодаря этой документации вы можете улучшить систему отчетности GDPR, а также понять технические и организационные меры. Документация по оценкам влияния на защиту данных, запросам на тему данных и уведомлению о нарушении данных предоставляется для внедрения в собственную программу поддержки GDPR.
  - **Запросы субъекта данных:** GDPR предоставляет лицам (или темам данных) определенные права в связи с обработкой их персональных данных. Эти права включают право на исправление неточных данных, удаление данных или ограничение их обработки, а также право на получение данных и выполнение запроса на передачу данных на другой контроллер.
  - **Нарушение данных:** GDPR требует требования к уведомлениям для контроллеров данных и процессоров, если происходит нарушение персональных данных. Портал доверия служб Майкрософт предоставляет сведения о том, как работает корпорация Майкрософт, чтобы предотвратить нарушения, как корпорация Майкрософт обнаруживает нарушение, а также сообщает о том, как будет реагировать и уведомлять вас в качестве контроллера данных в случае возникновения нарушения.
  - **Оценка влияния на защиту данных:** Корпорация Майкрософт помогает контроллерам выполнять оценку влияния на защиту данных GDPR (Дпиас). GDPR предоставляет неисчерпывающий список вариантов, в которых необходимо выполнить Дпиас, например автоматическую обработку в целях профилирования и выполнения аналогичных действий. обработка большого масштаба специальных категорий персональных данных и систематический мониторинг общедоступной области в больших масштабах.
  - **Другие ресурсы:** В дополнение к средствам, описанным в разделах выше, портал доверия служб Майкрософт также предоставляет другие ресурсы, включая региональные соответствия, дополнительные ресурсы для центра безопасности и соответствия требованиям, а также часто задаваемые вопросы о портале доверия служб Майкрософт, диспетчере соответствия требованиям, конфиденциальности и GDPR.
- **Региональное соответствие:** Портал доверия служб Майкрософт предоставляет многочисленные документы о соответствии и рекомендации для веб-службы Майкрософт в соответствии с требованиями к стандартам для различных регионов, включая Чешскую Республика, Польша и Румыния.

## <a name="azure-threat-intelligence"></a>Аналитика угроз Azure

С помощью функции "Анализ угроз", доступной в центре безопасности, ИТ-администраторы могут выявлять угрозы безопасности в среде. Например, определять, что конкретный компьютер является частью ботнета. Компьютеры могут стать узлами в ботнете, если злоумышленники незаконно установят вредоносную программу, которая тайно подключает этот компьютер к среде управления и контроля. С помощью анализа угроз можно выявить потенциальные угрозы, поступающие по нелегальным каналам связи, таким как даркнет.

Для создания этой аналитики угроз центр безопасности использует данные из нескольких источников в корпорации Майкрософт. Центр безопасности использует эти данные для выявления потенциальных угроз в вашей среде. Панель "аналитика угроз" состоит из трех основных параметров:

- Типы обнаруженных угроз
- Источник угроз.
- Карта анализа угроз

## <a name="machine-learning-in-azure-security-center"></a>Машинное обучение в Центре безопасности Azure

Центр безопасности Azure глубоко анализирует множество данных от различных решений Майкрософт и партнеров, чтобы помочь вам повысить безопасность. Чтобы воспользоваться преимуществами этих данных, компания использует обработку и анализ данных и машинное обучение для предотвращения угроз, их обнаружения и, в конечном итоге, исследования.

В широком смысле Машинное обучение Azure позволяет достигнуть двух результатов, которые представлены ниже.

### <a name="next-generation-detection"></a>Обнаружение следующего поколения

Злоумышленники становятся все более автоматизированными и изощренными. Они тоже используют обработку и анализ данных. Они реконструируют защиту и создают системы, которые поддерживают мутации в поведении. Они маскируют свои действия под шум и быстро учатся на ошибках. Машинное обучение помогает нам реагировать на эти разработки.

### <a name="simplified-security-baseline"></a>Упрощенный базовый уровень безопасности

Принимать эффективные решения по безопасности непросто. Для этого необходим опыт и знания в области безопасности. Хотя некоторые крупные организации обладают такими экспертами по персоналу, многие компании их не используют. Машинное обучение Azure позволяет клиентам использовать решения других организаций при принятии решений в области безопасности.

## <a name="behavioral-analytics"></a>Поведенческая аналитика

Поведенческая аналитика — это метод, в рамках которого данные анализируются и сверяются с набором известных схем. Эти шаблоны не являются простыми подписями. Они определяются с помощью сложных алгоритмов машинного обучения, применяемых к огромным наборам данных. Они также определяются с помощью тщательного анализа вредоносных поведений экспертами. Поведенческая аналитика позволяет центру безопасности Azure выявлять скомпрометированные ресурсы. Для этого центр безопасности анализирует журналы виртуальных машин, журналы устройств виртуальной сети, журналы структуры, аварийный дампы и другие источники.
